# Google Sheets Helper - google 表格工具

## 介绍
该仓库实现了一些基于 Google Sheets 表格 Apps Script 的工具类功能，主要专注于解决将 Goole Sheets 表格作为配置表需求的业务处理方案。

## 功能
- [ ] Base64 编解码函数
- [ ] 支持两个表格页签的数据对比

## 安装及使用方法
1. 打开 Google Sheets
2. 点击 "Extensions" > "Apps Script"
3. 复制以下文件到对应位置：
   - Code.gs
   - Compare.gs
   - CompareDialog.html
4. 保存后刷新界面
