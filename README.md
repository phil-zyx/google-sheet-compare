   # Google Sheets 表格比较工具

   ## 功能特点
   - 支持两个表格的数据对比
   - 智能识别表头差异
   - 可视化展示数据变更
   - 支持批量数据处理

   ## 安装方法
   1. 打开 Google Sheets
   2. 点击 "工具" > "脚本编辑器"
   3. 复制以下文件到对应位置：
      - Code.gs
      - Compare.gs
      - CompareDialog.html

   ## 使用说明
   1. 安装完成后刷新表格
   2. 在菜单栏找到"表格比较工具"
   3. 选择要比较的表格
   4. 查看比较结果

   ## 注意事项
   - 需要具备表格编辑权限
   - 建议在重要数据操作前备份
   - 表头差异会以黄色标记
